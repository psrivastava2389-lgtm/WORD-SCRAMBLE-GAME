# WORD SCRAMBLE GAME
A word scramble game built using Python and MySQL where users guess the correct word from scrambled letters.
<br>
## Features
- Random word generation
- Score tracking
- MySQL database integration
<br>
## Tech Stack
- Python
- MySQL
<br>
## Setup Instructions
1. Clone the repo
2. Install dependencies
3. Import word_scramble.sql into MySQL
##DATABASE SETUP
## 🗄️ Database Setup (MySQL)

3.1. Open MySQL (Command Line or MySQL Workbench)

3.2. Create a database and import data of word_scramble to your database:

"CREATE DATABASE your_database;
USE your_database;
source word_scramble.sql;;"

4. Run word_scramble.py
<br>
## How to Run
python word_scramble.py
